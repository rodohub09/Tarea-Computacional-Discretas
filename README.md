# Instrucciones de compilación
### Instalar la librería GLib en caso de no tenerla
```bash
sudo apt update
sudo apt install libglib2.0-dev
```
### Compilar el código mediante makefile
```bash
make main
```
### Ejemplo de ejecución 
```bash
./path a b arbol_grafo.txt
```
